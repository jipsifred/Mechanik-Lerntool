# Mechanik Lerntool

AI-powered physics tutoring app for d'Alembert's principle problems.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Generate local config (JWT secrets etc.):
   `npm run setup`
3. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
4. Run the app:
   `npm run dev:all`
